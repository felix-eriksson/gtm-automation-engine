# Media Generation

This folder represents automated media asset generation used to power highly personalized GTM outreach.

Business goals of this subsystem:
- Generate prospect-specific visual assets (logos, brand colors, profile images)
- Personalize outbound videos and creative at scale
- Increase engagement by embedding familiar brand and identity signals
- Eliminate manual creative work in outbound campaigns

This subsystem enables one-to-one personalization while operating at one-to-many scale.
